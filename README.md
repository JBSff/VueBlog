# v1.0 更新内容

相对于旧版本，本项目包含以下更新：

## ✨ 新增特性

- **数据持久化**: 使用 LocalStorage 本地存储数据，刷新页面数据不丢失。
- **状态管理增强**: 深度集成 Pinia 进行状态管理。
- **完整交互体验**: 结合 LocalStorage 和 Mock 数据，模拟了完整的前后端交互流程。

## 🚀 环境要求

- Node.js >= 16.0.0

## 📝 说明

本项目主要用于演示 Vue 3 生态系统的使用。数据默认存储在浏览器的 LocalStorage 中。如果需要重置数据，可以清除浏览器缓存或手动清空 LocalStorage。
