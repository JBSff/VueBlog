# 个人博客系统 v2.0 (Personal Blog System)

## 📅 更新日志 (Changelog)

### v2.0
- **结构优化**: 移除了冗余的测试组件 (`TestPersistenceView`)，精简了项目结构。
- **路由更新**: 优化了路由配置，移除了测试路由，提升了代码整洁度。
- **体验提升**: 修复了潜在的路由引用错误，确保应用启动更加稳定。
- **持久化升级**: 引入 `pinia-plugin-persistedstate` 插件，替代手动封装的 `localStorage` 工具，实现更优雅的状态持久化。
- **代码清理**: 删除了不再使用的工具类文件 (`src/utils/storage.js`) 和未引用的资源文件。
- **目录调整**: 将用户个人中心视图 (`UserProfileView`) 移动至后台目录 (`backend`)，规范了文件组织。
