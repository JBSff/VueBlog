# VueBlog 更新日志

## v1.4 (2025-12-07)

### 核心更新

#### 1. 部署配置优化
- **GitHub Pages 适配**: 更新 `vite.config.js`，根据环境动态配置 `base` 路径，完美支持 GitHub Pages 部署。
- **一键部署脚本**: 在 `package.json` 中集成 `deploy` 脚本，支持通过 `npm run deploy` 命令自动构建并推送到 `gh-pages` 分支。

#### 2. 管理后台功能增强
- **文章搜索重构**: 
  - 废弃了不稳定的手动监听模式，采用 Vue `computed` 计算属性实现实时搜索。
  - 支持多维度过滤：现在可以同时搜索文章标题、内容、分类及作者。
  - 修复了输入搜索关键词后列表无反应的 Bug。
- **阅读量数据展示**: 
  - 修复了管理后台文章列表中“阅读量”列显示为空的问题。
  - 统一了数据字段映射，确保 `viewCount` 正确渲染。

#### 3. 系统稳定性与修复
- **管理员权限持久化**: 
  - 引入用户角色归一化逻辑，修复了 Admin 账号在页面刷新后意外降级为普通用户的严重 Bug。
  - 优化了 `userStore` 的状态恢复机制。
